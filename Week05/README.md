学习笔记

Proxy重新建立一个可以拦截改变对象默认行为的对象，增大了代码的不可预测性。
vue的reactivity实现通过调用这个回调触发get来检测函数使用了哪些的值,把值和回调保存起来，那么有set的话就会调用回调，回调中获取到也就是最新的值了。
range 防止拖动脱落，鼠标弹起和移动事件放在document上，移出浏览器也可以监听。
使元素在正常流移动，使用translate和行内块。
使用range精确控制插入DOM位置。
